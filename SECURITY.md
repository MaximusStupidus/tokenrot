# Security & privacy

`vibeaudit` runs entirely on your machine, has zero runtime dependencies, and makes no network
requests. Your code and prompts are never read or transmitted (see [docs/PRIVACY.md](docs/PRIVACY.md)).

## Reporting
Found a security or privacy issue (e.g. anything that could cause data to leave the machine)? Please
open a GitHub issue marked `[security]`, or contact the maintainer directly rather than posting
exploit details publicly. We'll respond promptly.
